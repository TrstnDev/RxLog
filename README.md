# RxLog

> A clinical logging app for healthcare workers.

[![CI](https://github.com/TrstnDev/RxLog/actions/workflows/ci.yml/badge.svg)](https://github.com/TrstnDev/RxLog/actions/workflows/ci.yml)
![Swift](https://img.shields.io/badge/Swift_6-F05138?logo=swift&logoColor=white)
![Xcode](https://img.shields.io/badge/Xcode_26.5-147EFB?logo=xcode&logoColor=white)
![iOS](https://img.shields.io/badge/iOS_26.0%2B-000000?logo=apple&logoColor=white)

RxLog is a native iOS app for capturing patient histories, notes, observations, and handovers quickly and privately. It's designed around Apple's Human Interface Guidelines and the iOS 26 **Liquid Glass** design language.

> **Note:** RxLog is an active work in progress.
